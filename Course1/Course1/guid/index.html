Учебная игра по параллельному программированию

Руководство пользователя

В игре нужно написать алгоритм для группы дронов так, чтобы они уничтожили все сорняки на карте. Команды выполняются по тикам: внутри одного тика несколько дронов могут действовать параллельно.

Команды Карты Алгоритмы

Цель игры

На карте находятся дроны и сорняки. Пользователь заполняет таблицу команд, запускает алгоритм и проверяет, сможет ли алгоритм уничтожить все сорняки без ошибок.

Успешное завершение

Все сорняки уничтожены, столкновений нет, дроны не вышли за границы карты.

Ошибка

Дрон врезался в границу, столкнулся с другим дроном, атаковал пустую клетку или алгоритм не уничтожил все сорняки.

Как писать команды

Таблица команд состоит из тиков. Один тик — это один шаг алгоритма. Команды, записанные в одном тике, выполняются параллельно.

Адресат Действие Аргумент Описание
Все Вперёд 1, 2, 3... Все дроны делают указанное количество шагов вперёд.
Дрон 1 Налево пусто или 1 Дрон поворачивается на 90° влево.
Дрон 2 Направо пусто или 1 Дрон поворачивается на 90° вправо.
Дрон 3 Разряд пусто или 1 Дрон уничтожает сорняк в клетке перед собой.
1

Выберите адресата

Можно выбрать конкретного дрона или команду «Все».

2

Выберите действие

Доступны движения, повороты и атака.

3

Запустите алгоритм

Нажмите «Выполнить» и дождитесь результата.

Параллельное выполнение

Если в одном тике заданы команды для нескольких дронов, они начинают выполняться одновременно. Для карт с большим количеством дронов тик может занимать несколько строк таблицы. 

Тик 1: Дрон 1 — Вперёд    Дрон 2 — Вперёд
Тик 1: Дрон 3 — Разряд    Дрон 4 — Налево
Тик 2: Все    — Вперёд

В примере первые четыре команды относятся к одному тику, а команда «Все — Вперёд» выполнится уже следующим шагом.

Оценка алгоритма

После успешного завершения игра показывает оценку алгоритма.

Оценка = повороты налево/направо + команды «Вперёд» × 2

Команда «Разряд» не увеличивает оценку. Чем ниже оценка, тем короче и эффективнее алгоритм.

Как создавать карты

Откройте меню Файл → Открыть редактор карт. В редакторе можно выбирать клетки и добавлять или удалять дронов и сорняки.

  • На карте должен быть хотя бы один дрон.
  • Дронов не может быть больше 10.
  • Количество сорняков не должно быть меньше количества дронов.
  • Дрон и сорняк нельзя поставить в одну и ту же клетку.
  • Карта сохраняется в папку levels в формате JSON.
{
  "drones": [
    { "x": 2, "y": 5 },
    { "x": 10, "y": 8 }
  ],
  "weeds": [
    { "x": 3, "y": 5 },
    { "x": 4, "y": 5 },
    { "x": 11, "y": 8 }
  ]
}

Как сохранять и загружать алгоритмы

Команды из таблицы можно сохранить через Файл → Сохранить алгоритм. Алгоритмы хранятся в папке algorithms в формате JSON.

  • При сохранении укажите название алгоритма.
  • При загрузке выберите JSON-файл из папки algorithms.
  • Алгоритм содержит количество дронов, для которого он был создан.
  • Если количество дронов на текущей карте другое, алгоритм не загрузится.
{
  "numberDronesOnMap": 2,
  "ticks": [
    {
      "commands": [
        { "target": "drone", "droneNumber": 1, "action": "forward", "repeat": 1 },
        { "target": "drone", "droneNumber": 2, "action": "attack", "repeat": 1 }
      ]
    }
  ]
}

Советы

Планируйте направление. Команда «Разряд» действует на клетку перед дроном, поэтому повороты важны.

Проверяйте тики. Если несколько дронов должны действовать одновременно, помещайте команды в один тик.

Следите за зарядами. Сумма зарядов дронов равна количеству сорняков, поэтому лишние атаки приведут к ошибке.

Используйте тестовые алгоритмы. Они помогают проверить успешное прохождение, столкновение, выход за карту и неполное уничтожение сорняков.
